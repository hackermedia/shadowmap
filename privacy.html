Privacy Policy for Shadow Map

Shadow Map is designed with privacy as a core principle.

No Data Collection

Shadow Map does not collect, store, transmit, or sell any personal data.

All network visualization and analysis performed by Shadow Map occurs locally within the user’s browser. No browsing history, website content, or network request data is sent to any external servers.

No Tracking or Analytics

Shadow Map does not use analytics, telemetry, tracking scripts, or user behavior monitoring of any kind.

Local Processing Only

Any network connection information displayed by Shadow Map is processed temporarily in memory for visualization purposes and is discarded when the page or browser session ends.

Third Party Services

Shadow Map does not integrate with third party analytics, advertising platforms, or data brokers.
